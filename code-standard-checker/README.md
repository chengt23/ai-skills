# 代码规范检查技能

这是一个用于检查代码规范的工具，支持多种编程语言的规范检查。

## 功能特性

- **命名规范检查**：检查类名、方法名、变量名、常量名是否符合规范
- **注释规范检查**：检查代码注释是否完整、是否使用行尾注释等
- **安全问题检测**：检测SQL注入、硬编码密码、XSS等安全问题
- **性能问题检测**：检测循环中的数据库操作、未指定容量的集合等性能问题
- **代码风格检查**：检查代码格式、风格是否符合项目规范

## 支持的语言

- Java
- JavaScript/TypeScript
- Python
- HTML/CSS

## 使用方法

```bash
# 检查单个文件
python main.py <file_path>

# 示例
python main.py ./src/main/java/com/example/UserService.java
```

## 输出格式

工具会生成详细的检查报告，包括：

- 文件信息
- 总体评分
- 问题汇总表格
- 详细分析

## 严重程度定义

- **高**：可能导致系统错误、安全漏洞或严重性能问题
- **中**：影响代码可读性、可维护性，或存在潜在风险
- **低**：轻微的风格问题，建议改进但不强制

## 技术实现

基于Python实现的静态代码分析工具，通过正则表达式和语法模式匹配来检测代码中的问题。
